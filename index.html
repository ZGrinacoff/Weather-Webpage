Summary: Latitude vs. X...


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this, we first assembled a dataset on over 500 cities from the OpenWeatherMap API.


After assembling the dataset, we used Pandas and Matplotlib to compare various aspects of the weather. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
